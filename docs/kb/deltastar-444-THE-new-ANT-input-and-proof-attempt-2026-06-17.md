## Part 3 — Crossing the bridge: new mathematics (the genuine attempt)

### Idea A — the Frobenius / decomposition-group orbit principle [NEW; partial]
**Observation [PROVEN].** Fix a prime `𝔭 | p` in `ℤ[ζ_n]`. The decomposition group `D_𝔭 = ⟨σ_p⟩` (Frobenius
`ζ ↦ ζ^p`, order `f = ord_n(p)`) **fixes `𝔭`**, hence permutes the wraparound set `S_r` (the condition
`Σx ≡ Σy mod 𝔭` is `σ_p`-stable, and the char-0 condition `Σx=Σy` is Galois-stable). Therefore
**`W_r = |S_r|` is a sum of `D_𝔭`-orbit sizes**, each dividing `f`.

**Structure for `n=2^μ` [MACHINE-VERIFIED `(ℤ/2^μ)^× = ℤ/2 × ℤ/2^{μ−2}`].** Element orders are **powers of
2**; `f = 2^s`. The unique order-2 element of `D_𝔭` is `τ = σ_p^{f/2} : ζ↦ζ^c`, `c ∈ {−1, 2^{μ−1}±1}`.
- If `τ = ` complex conjugation (`c=−1`, i.e. `−1 ∈ ⟨p⟩ mod n`): conjugation-fixed wraparound solutions
  (`{x_i},{y_j}` closed under `x↦x⁻¹`, so `Σx,Σy ∈ ℝ`) can exist — the orbit principle gives only
  `W_r ≡ 0 (mod 2)` (every non-fixed orbit even). **[the data has `W_3=0`, consistent.]**
- **If `−1 ∉ ⟨p mod n⟩` (conjugation NOT in the decomposition group):** the wraparound solutions carry NO
  real/antipodal symmetry; their orbits are governed by the full `2^s`-structure. **CONJECTURE A:** for
  this positive-density class of primes (a Chebotarev condition on `p mod 2^μ`), every wraparound orbit
  has size `≥ 4`, forcing `4 | W_r`, and — combined with a depth-`r` *bound* `W_r < f = 2^s` from a
  separate count — `W_r = 0`. **Status:** the orbit principle is PROVEN; the size-`≥4` claim and the
  `W_r < f` bound are the open pieces. This is genuinely new (no prior Frobenius-orbit analysis of the
  wraparound) and reduces the prize to a *Chebotarev class + an orbit-size lower bound* — a sharper,
  more structured target than "bound a character sum."

### Idea B — the master-resultant good-prime density [NEW framing of the Linnik residual]
`W_r = 0` for all `r ≤ R` ⟺ `p ∤ R_n(R)`, where `R_n(R) = ∏ N(δ)` over the finitely-many distinct nonzero
